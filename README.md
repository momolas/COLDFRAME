# COLDFRAME (Al Qibla)

COLDFRAME is a modern iOS application designed to help users find the Qibla direction and view accurate prayer times. Built with SwiftUI and Swift 6.2.

## Features

- **Qibla Compass**: High-precision compass pointing to the Kaaba (Mecca).
- **Haptic Feedback**: Haptic confirmation when the device is aligned with the Qibla.
- **Prayer Times**: Accurate prayer times calculated locally based on your geographical location.
- **Modern UI**: Dark-themed user interface with gold accents, using latest SwiftUI standards.
- **Privacy Focused**: Location data is processed entirely on-device.

## Requirements

- iOS 26.0+
- Swift 6.2+

## Installation

1. Open `COLDFRAME.xcodeproj` in Xcode.
2. Wait for package dependencies (SwiftAA) to resolve.
3. Build and run on your device.
4. Grant location access when prompted to enable compass and prayer time features.
